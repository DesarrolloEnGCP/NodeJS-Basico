# NodeJS Basico

## Comprobar version de NodeJS
```bash
node --version
```

## Comprobar version de npm
```bash
npm --version
```

## Crear un nuevo directorio
```bash
mkdir testnodejs || cd testnodejs
```

## Crear un nuevo proyecto
```bash
npm init
```
